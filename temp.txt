# AI新媒体矩阵网站开发记录

## 2024-03-27

### 初始需求
我想做一个网站，用于介绍我们的多个产品。首页是一个目录，可以参考@https://ai.5ienet.com/aihospital/index.html的样式，每行目录即为每个产品页面独立的跳转入口。

要求网站具备：
1、完全响应式设计，适配各种屏幕尺寸
2、使用ECharts实现丰富的数据可视化
3、现代化的UI设计，包含动画效果
4、清晰的导航结构

技术栈：
- HTML5
- CSS3 (使用CSS变量和Grid/Flexbox布局)
- JavaScript (原生JS)
- ECharts (数据可视化)
- Font Awesome (图标库)
